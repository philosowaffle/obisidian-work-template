# Shortcuts

| Description              | Command                            |
| ------------------------ | ---------------------------------- |
| Close Tab                | `Cmd + W`                          |
| Go To                    | `Cmd + T`                          |
| Open Settings            | `Cmd + ,`                          |
| Nav Back/Forward         | `Ctrl + [ ]`                       |
| New note from template   | `Cmd + Shft + N`                   |
| Open todays Daily note   | `Cmd + H`                          |
| Toggle Side Panels       | `Ctrl + Alt + left/right arrow`    |
| Next Tab / Prev Tab      | `Ctrl + Tab` / `Ctrl + Shft + Tab` |
